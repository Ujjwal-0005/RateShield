# RateShield

RateShield is a production-grade distributed rate limiter built with JavaScript, Node.js, Express.js, Redis, React, Docker, and Redis Cloud.

## Current Phase

Phase 1 - Project Initialization

## Workspace Layout

- `backend` contains the Express API foundation.
- `frontend` contains the React dashboard scaffold.
- `docs` contains the project roadmap.
- `screenshots` contains reference images for the UI and implementation notes.

## Initialization Scope

This phase focuses on the code and structure needed to start the project cleanly.

- Backend package metadata and scripts.
- Environment-based configuration.
- Express app bootstrap.
- Route, controller, and middleware scaffolding.
- A basic health endpoint for readiness checks.

No rate-limiting business logic is added yet. That belongs to later phases so the architecture stays easy to change and test.
